EXPLAINING THE DATA¶

TRAIN DATA¶

  1. The train data has 8693 enteries(rows), and 14 features(columns)

  2. It has 7 categoical features('HomePlanet','CryoSleep','Cabin','Destination','VIP','Name'), and 7 numerical features('Age', 'RoomService', 'FoodCourt', 'ShoppingMall', 'Spa', 'VRDeck', 'Transported')

  3. The train data has 26.8% of values missing. The features with missing values are; 'HomePlanet','CryoSleep','Cabin','Destination','VIP','Name', 'Age', 'RoomService', 'FoodCourt', 'ShoppingMall', 'Spa', 'VRDeck'

  4. There are 3 Home Planets: Earth, Europa and Mars. Also there are three destinations according to the data: TRAPPIST-1e, 55 Cancri e, and PSO J318.5-22

  5. The train data's categorical features have there modes to be: HomePlanet=Earth(4602 times), CryoSleep=False(5439 times), Destination=TRAPPIST-1e(5915 times),VIP=False(8291 times), Cabin=G/734/S(8 times)

TEST DATA¶

  1. The test data has 4277 enteries(rows), and 13 features(columns)

  2. It has 7 categoical features('HomePlanet','CryoSleep','Cabin','Destination','VIP','Name'), and 6 numerical features('Age', 'RoomService', 'FoodCourt', 'ShoppingMall', 'Spa', 'VRDeck')

  3. The test data has 26.2% of values missing. The features with missing values are; 'HomePlanet','CryoSleep','Cabin','Destination','VIP','Name', 'Age', 'RoomService', 'FoodCourt', 'ShoppingMall', 'Spa', 'VRDeck'

  4. There are 3 Home Planets: Earth, Europa and Mars. Also there are three destinations according to the data: TRAPPIST-1e, 55 Cancri e, and PSO J318.5-22

  5. The test data's categorical features have there modes to be: HomePlanet=Earth(2263 times), CryoSleep=False(2640 times), Destination=TRAPPIST-1e( 2956 times),VIP=False(4110 times), Cabin=G/160/P(8 times)

Home Planet¶

Destination¶
